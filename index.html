Point Color

메인페이지는 카테고리별로 상품정보를 제공합니다.

서브페이지 또한 jQuery를 활용하여 짜임새 있는 구성으로
도서연관 정보를 제공하고있습니다.


사용기능 : openAPI, JSON, Ajax jQuery, Html, CSS
